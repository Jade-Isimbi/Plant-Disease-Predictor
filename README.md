# Plant Disease Classification - ML Pipeline

A complete Machine Learning pipeline for plant disease classification using the PlantVillage dataset, deployed on cloud infrastructure.

##  Project Description

This project implements ML pipeline for classifying plant diseases from images. The system includes:
- Data acquisition and preprocessing
- Model training and evaluation
- RESTful API for predictions
- Web UI for monitoring, visualizations, and retraining
- Cloud deployment with Docker
- Load testing with Locust

##  Dataset

### PlantVillage Dataset

This project uses the [PlantVillage Dataset](https://www.kaggle.com/datasets/abdallahalidev/plantvillage-dataset) from Kaggle.
